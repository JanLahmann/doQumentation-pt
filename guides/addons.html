Addons do Qiskit

Os addons do Qiskit são uma coleção de capacidades de pesquisa para possibilitar a descoberta de algoritmos em escala utilitária. Essas capacidades se constroem sobre a base de ferramentas de alto desempenho do Qiskit para criar e executar algoritmos quânticos. Elas são fornecidas como componentes de software modulares que podem ser integrados a um fluxo de trabalho para escalar ou projetar novos algoritmos quânticos.

Addons para mapeamento​

Compilação quântica aproximada com redes tensoriais​

A compilação quântica aproximada com redes tensoriais (AQC-Tensor) permite a construção de circuitos de alta fidelidade com profundidade reduzida.

  • Acesse o repositório GitHub.
  • Leia a documentação.
  • Leia o tutorial sobre como usar o AQC para melhorar a evolução temporal Trotterizada.

Fórmulas multi-produto​

As fórmulas multi-produto (MPF) reduzem o erro de Trotter na dinâmica hamiltoniana por meio de uma combinação ponderada de várias execuções de circuitos.

  • Acesse o repositório GitHub.
  • Leia a documentação.

Mapeador de otimização​

O addon Optimization Mapper contém funcionalidades para modelar problemas de otimização, formulando-os em modelos abstratos e convertendo-os em representações que um computador quântico consegue entender.

  • Leia a documentação para guias práticos e referências de API.
  • Acesse o repositório no GitHub.

Addons para otimização​

Retropropagação de operadores​

A retropropagação de operadores (OBP) reduz a profundidade do circuito removendo operações do final, ao custo de mais medições de operadores.

  • Acesse o repositório GitHub.
  • Leia a documentação.
  • Leia o tutorial sobre como usar o OBP para melhorar valores de expectativa.

Corte de circuitos​

O corte de circuitos reduz a profundidade dos circuitos transpilados decompondo portas de entrelaçamento entre qubits não adjacentes.

  • Acesse o repositório GitHub.
  • Leia a documentação.

Addons para pós-processamento​

Diagonalização quântica baseada em amostras​

A diagonalização quântica baseada em amostras (SQD) realiza o pós-processamento clássico de amostras quânticas ruidosas para obter estimativas mais precisas de autovalores de hamiltonianos de sistemas quânticos, como em aplicações de química.

  • Acesse o repositório GitHub.
  • Leia a documentação.
  • Leia o tutorial sobre diagonalização quântica baseada em amostras de um hamiltoniano de química.

SQD para HPC​

Este addon é uma implementação do addon SQD preparada para HPC. Ele é escrito seguindo os padrões modernos do C++17 e foi projetado para criar um único binário compilado para uso com MPI.

  • Acesse a documentação de referência da API.
  • Veja o repositório no GitHub.

Addons para mitigação de erros​

M3 (mthree)​

A Mitigação de Erros de Medição sem Matriz (M3) é um pacote para mitigação escalável de erros de medição quântica que pode ser computado em paralelo.

  • Acesse a documentação para guias práticos, tutoriais e referências de API.
  • Acesse a página do pacote no GitHub.

Cones de luz sombreados​

O addon de cones de luz sombreados utiliza a propagação de Pauli para reduzir o número de termos de erro em um modelo de ruído que precisam ser mitigados, de acordo com as especificidades do observável alvo. Isso tem o efeito de reduzir a sobrecarga de amostragem em fluxos de trabalho de cancelamento probabilístico de erros (PEC).

  • Acesse a documentação para guias práticos, tutoriais e referências de API.

Absorção de ruído propagado​

A absorção de ruído propagado (PNA) utiliza a propagação de Pauli para absorver informações de um modelo de ruído em um observável alvo. Medir esse observável modificado tem o efeito de mitigar o ruído conforme representado pelo modelo.

  • Acesse a documentação para guias práticos, tutoriais e referências de API.
