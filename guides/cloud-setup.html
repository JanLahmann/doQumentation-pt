Configure sua conta IBM Cloud

Se você recebeu um convite por e-mail para entrar em uma conta, siga as etapas em Configure sua conta IBM Cloud - usuários convidados.

Você pode executar cargas de trabalho em unidades de processamento quântico (QPUs) da IBM® configurando uma conta no IBM Cloud®. Você também precisará instalar e configurar o Qiskit e o Qiskit Runtime.

Sua conta de usuário está associada a uma ou mais instâncias, identificadas por Cloud Resource Names (CRNs) que fornecem acesso aos serviços IBM Quantum. Além disso, um token de API exclusivo (também chamado de chave) é atribuído a cada conta, permitindo o acesso ao IBM Quantum pelo Qiskit. Para instruções sobre como escolher uma instância específica, consulte Conectar-se a uma instância.

Configurar para usar o IBM Cloud​

Antes de começar​

  1. Se você ainda não tiver uma, crie uma conta IBM Cloud.

    nota

    Você pode fazer parte de várias contas IBM Cloud. Você pode acessar qualquer uma das suas contas IBM Cloud a qualquer momento pelo seletor de contas no cabeçalho da interface do IBM Quantum Platform.

  2. Faça login no IBM Quantum Platform com um IBMid ou conta Google. Se você não tiver um, será guiado pelo processo de criação. (Essas credenciais de login não são as credenciais da sua conta IBM Cloud.)

  3. Certifique-se de que a conta e a região corretas estão selecionadas no seletor de contas no cabeçalho, conforme mostrado na imagem a seguir.

    A região controla onde seus jobs são executados e onde os dados dos jobs são armazenados. Você pode acessar qualquer região usando a mesma chave de API, mas só pode ver e acessar as instâncias criadas na região em que você está conectado.

1. Criar uma instância​

Você pode criar uma instância na página Instâncias do IBM Quantum Platform. Consulte Criar uma instância para obter instruções detalhadas.

2. Opcional: Salvar suas credenciais de acesso​

Se você estiver usando um ambiente Python confiável, como um laptop pessoal, é recomendável usar o método save_account() para salvar suas credenciais localmente.

3. Conectar o Qiskit à sua instância de serviço​

  • Inicialize o serviço em um ambiente Python confiável
  • Inicialize o serviço em um ambiente não confiável (como um computador público)
Notas
  • Siga estas instruções se quiser conectar usando a REST API em vez do Qiskit.
  • Se necessário, use estas informações para configurar seu firewall e habilitar o acesso aos endpoints da API IBM Quantum.
